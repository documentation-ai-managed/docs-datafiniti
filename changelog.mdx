<Callout kind="alert" title="Migration Guide">
For v1.0.0 breaking changes, update your API calls to use the new `/v1` prefix. Test in staging first.

```bash
# Old endpoint (deprecated)
curl https://api.example.com/v0/search?q=products

# New endpoint
curl https://api.example.com/v1/search?q=products
```
