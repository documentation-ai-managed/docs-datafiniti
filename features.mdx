## Overview

Datafiniti provides powerful tools for managing your documentation projects. You can organize content efficiently, collaborate with teams, search quickly, export in various formats, and customize your spaces to match your brand. These features help you maintain professional documentation at scale.

## Document Organization and Folders

Organize your documentation using nested folders. Create a main folder for projects, then add subfolders for guides, APIs, and FAQs.

### Quick Setup Steps

<Steps>
  <Step title="Create Folder" icon="folder-plus">
    Navigate to your space and click the new folder button.
  </Step>
  <Step title="Add Documents" icon="file-plus">
    Drag and drop MDX files or create new pages directly.
  </Step>
  <Step title="Nest Subfolders" icon="chevron-down">
    Right-click a folder to create subfolders for deeper organization.
  </Step>
</Steps>

## Version Control and Collaboration

Datafiniti includes built-in version control. You and your team can edit simultaneously, with automatic conflict resolution.

<CodeGroup tabs="CLI,API">
  ```bash
  # Pull latest changes
  datafiniti pull --space my-docs

  # Push your updates
  datafiniti push --message "Add features page"
  ```
  ```javascript
  // Fetch space via API
  const response = await fetch('https://api.example.com/v1/spaces/my-docs/versions/latest', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const data = await response.json();
  ```
</CodeGroup>

<Callout kind="tip">
Enable real-time collaboration by inviting team members to your space.
</Callout>

## Search and Tagging Functionality

Tag documents with keywords like `api`, `guide`, or `internal`. Use the global search bar for full-text queries.

| Feature | Description | Example Tags |
|---------|-------------|--------------|
| Basic Search | Find by title or content | `auth`, `setup` |
| Advanced Filters | Combine tags and dates | `api + updated>2024` |
| Saved Searches | Bookmark frequent queries | `weekly-reports` |

<Tabs>
